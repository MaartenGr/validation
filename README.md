# Validation Techniques
This repo contains the code for the validation blog.

Subjects include:
* Train/test split
* k-Fold CV
* Leave-one-out CV
* Leave-one-group-out CV
* Time-series CV
* Nested CV
* Wilcoxon signed-rank test
* McNemar's test
* 5x2CV paired t-test
* 5x2CV combined F test
